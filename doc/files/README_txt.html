vlad-push

  • github.com/jmervine/vlad-push

DESCRIPTION:

Adds support to Vlad for pushing files to a remote server without SCM.

FEATURES/PROBLEMS:

  • Requires that remote server has git installed
  • Requires vlad-git gem

SYNOPSIS:

  $ rake <env> vlad:push

REQUIREMENTS:

  • Vlad
  • Vlad-git

INSTALL:

    $ sudo gem install vlad-push

VARIABLES:

repository:Vlad config param, should be set to a directory location to use as repo on remote server(s) :repository => "/tmp/repo"
domain:See Vlad :domain variable
scp_cmd:Defaults to scp
ssh_flags:See Vlad :ssh_flags variable
